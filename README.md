# Merca-Ya-Complete
MercaYA es un sistema web de gestión comercial para microempresas, desarrollado con React + TypeScript en el frontend y Node.js + Express + TypeScript en el backend. Permite administrar ventas, compras, inventario, clientes, cartera, caja, usuarios y despachos, con control de roles, seguridad mediante JWT y ejecución en entorno local
